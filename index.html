LRU Page Replacement Simulator

Visualize memory management step by step using the Least Recently Used algorithm.

See how memory pages are replaced using the Least Recently Used (LRU) algorithm. Enter a reference string and frames to watch hits, misses, and replacements step by step.

Important Info

  • Enter your reference string separated by spaces or commas (e.g., 7 0 1 2 or A,B,C).
  • Memory page determine the memory capacity. More pages = fewer page replacements.
  • Simulation shows Hit and Miss visually for each reference.
  • Total hits and misses are displayed at the bottom after simulation.
  • You can mix letters and numbers in your reference string.
